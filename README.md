# Atlas 🗺️

*Your life, organized.*

A personal life dashboard — schedule, gym progress, expenses (₪), goals, and household chores — in a single HTML file.

**Stack:** Vanilla HTML/JS · Supabase (auth + Postgres with row-level security) · Chart.js · GitHub Pages

Data is protected by Supabase Auth; every table has RLS so each account only sees its own rows.
